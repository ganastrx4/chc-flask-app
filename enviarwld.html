Sending to a Wallet Address

 

Important: You can only send crypto to a corresponding wallet that supports World Chain transactions. 

 

1. Go to the Wallet tab, then press the Send button

2. On the next page, select the token you wish to send

3. Type in the wallet address where you want to send tokens.

If it's a new address, you'll see New address. If it's been used before, you'll see Known address:

 

If you see Incorrect address appearing on screen, make sure to type in the correct address.

 

4. After entering the wallet address, select the amount you wish to send and press Continue

 

5. Review the transaction details, and then press the Confirm send button

6. Tap the Done button
